# v2.5.0 업데이트 — 과거 추세와 현재 호가의 실전 결합

## 핵심 변경

이번 버전은 단순 점수 혼합이 아니라 다음 순서로 판정합니다.

1. 15분·4시간·일봉에서 상승 정렬, 눌림, 회복, 혼조, 하락 상태를 구분합니다.
2. EMA·스윙 저점으로 과거의 구조적 지지를 찾습니다.
3. WebSocket 관찰 종료 시점의 최신 최우선 호가를 실제 진입 기준가격으로 사용합니다.
4. 충분한 실시간 표본에서만 반복 매수벽·매도벽·돌파 지지전환을 지지·저항에 반영합니다.
5. 구조적 지지 아래에 손절을 두고 실제 저항 앞에 목표를 둔 뒤 수수료·슬리피지 포함 손익비를 계산합니다.
6. 지지·저항·동적 표본 중 하나라도 확인되지 않으면 현재가 BUY를 금지합니다.
7. 눌림 대기는 눌림 가격을 기준으로 목표와 손절을 새로 계산하며, 현재가 계획을 재사용하지 않습니다.

## 데이터 범위

- 5분봉 144개: 약 12시간
- 15분봉 192개: 약 48시간
- 4시간봉 180개: 약 30일
- 일봉 200개: 약 200일

## 배포 확인

- 화면 왼쪽 위 엔진 버전이 `v2.5.0`인지 확인합니다.
- 후보 리포트에서 `가격 기준`, `구조적 지지`, `구조적 저항`, `현재가 기준 비용 포함 손익비`가 표시되는지 확인합니다.
- 동적 표본 부족 종목의 `동적 호가 안전성`이 PASS로 표시되지 않는지 확인합니다.
- 초기 티커와 실시간 가격이 다르면 실시간 괴리(bp)가 표시되는지 확인합니다.

## 주의

실시간 반복벽은 개별 주문 ID와 숨은 잔량을 볼 수 없으므로 확정적인 스푸핑·아이스버그 판정이 아닙니다. 충분한 표본과 실제 체결량을 결합한 회피·확인 신호로만 사용합니다.
